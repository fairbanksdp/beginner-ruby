# How to Ruby

## Variables
* Assign a value:
`var = 1`
* Use the variable:
`if var == 1`
* Types of variables:
  * String
  * Integer
  * Array
  * Float
  * Char
  * Boolean


## Conditionals
* These are if-else statements that operate based off of wheather or not the peramiters are true
```
if var==1
  do something
else
  do something else
end
```
* They can be embeded with in one another
```
if var==1
  if true
    do something
  end
end
```
